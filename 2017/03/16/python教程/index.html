python教程

　　Python是一种既强大而又易学的编程语言。它具有高效的高级数据结构和一个简单但有效的面向对象编程的方式。Python那优雅的语法和动态类型以及自然的语言解析，使它成为在许多领域的多数平台脚本处理和快速部署应用的理想语言。
　　在Python网站上的Python解释器和标准的扩展库均是免费的，它们或者以资源的方式或二进制文件的方式能够兼容所有主流的平台系统,而且都能免费部署。网站还包括很多免费的第三方Python组件、程序、工具和附加的文档说明。
　　Python解释器很容易扩展新功能并扩展嵌入使用C或者C++（或其他从C语言发展而来的语言）执行 的数据格式。Python也适于作为本地应用的扩展语言。
　　此教程向读者介绍Python语言系统的基本思想和特性。它有助于形成Python解释器使用的经验，所有的例子都是独立的，因此此教程也能够离线阅读。
　　标准组件和元件的说明参见Python标准库。Python语言参考中对该语言有更正式的定义。为了在C或C++中扩展Python可以参考Python解释器的扩展和嵌入和Python/C API参考手册。还有其他深入说明Python的书籍。
　　此教程不会全面地解释说明每一个特性甚至是每一个常用特性。相反，它会集中介绍Python中最具价值的特性，并将会给予你一个好的语言享受和风格。经此教程，你将能够读写Python组件和程序，你也将为阅读在Python标准库中更多丰富的Python标准库组件做好准备。
